Contact Book

This is a Java application that acts as a contact book. The program allows the user to add, search, delete, and view contact. The program also features the ability to save your contacts to a text file that it automatically loads on boot.

To-Do List Manager

This is a Java application that acts as a To-Do List Manager. The program allows the user to add tasks, remove tasks, mark tasks as completed, and view tasks. Within the remove, mark complete, and view tasks features, the user is able to search, sort, and filter through the lists of tasks. There is also a saving feature based off of login credentials so you can load an existing list and create a new one.
